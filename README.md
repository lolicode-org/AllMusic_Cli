已知问题：
* 启动后首次进入时图片可能会炸掉，退出重进就好了
* 图片加载失败后会连带着信息hud一起消失掉
* 有时播放的音乐或者显示的封面会是之前的歌的
* 以上问题主要发生在快速切歌时，使用时不频繁切歌能减少问题发生的概率。个人建议关掉封面显示（在模组设置里能找到）
